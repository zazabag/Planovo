# Планово — Демо: Учебные учреждения

Демонстрационная версия системы расписаний **Планово** для учебных учреждений (колледжи, техникумы, ВУЗы).

## Возможности

### 👨‍🎓 Представление ученика
- Просмотр расписания по группе
- Сетка расписания на неделю (десктоп) / по дням (мобильный)
- Индикатор текущего занятия "Сейчас"
- Переключение чётной/нечётной недели
- Цветовое кодирование предметов
- Бейджи типа занятия (Лекция/Практика)
- Бейджи совмещённых пар

### 👩‍🏫 Представление преподавателя
- Просмотр личного расписания
- Редактор доступности (указание слотов, когда преподаватель может работать)
- Копирование доступности между неделями
- Отправка данных учебной части

### 🏢 Интерфейс учебной части
- **Конструктор расписания**: пошаговое заполнение слотов (предмет/преподаватель/аудитория)
- **Автогенерация**: автоматическое создание расписания с учётом ограничений (по мотивам Planora)
- **Обзор**: статистика, конфликты, быстрый доступ к действиям
- Обнаружение конфликтов в реальном времени (преподаватель, аудитория, группа)
- Публикация расписания

## Технологии

- **Next.js 16** + App Router
- **TypeScript 5**
- **Tailwind CSS 4** + shadcn/ui
- **Zustand** (управление состоянием)
- Дизайн-система **Planovo** (indigo-purple gradient)

## Установка и запуск

```bash
# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
```

Открыть http://localhost:3000

## Источники вдохновения

- **ScheduleKEMS** — механики составления расписания, структура данных, интерфейс конструктора
- **Planora** — идея автогенерации расписаний на основе ограничений (constraint satisfaction)
- **Планово** — дизайн-система, цветовая палитра, визуальный стиль
