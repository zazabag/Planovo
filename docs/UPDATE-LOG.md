# Журнал обновлений — Планово

## Обновление от 16 июня 2025

### Что нового

#### Главная страница (Landing)
- Полноценный лендинг с hero-секцией "Планово — Расписание без хаоса"
- Секция "Возможности платформы" с 4 карточками
- Секция **"Наше решение"** с тремя интерактивными карточками-демо
- Кнопки "Открыть демо" для каждой ниши
- Адаптивная мобильная версия

#### Демо: Учебные учреждения
- 3 роли: Ученик, Преподаватель, Учебная часть
- Расписание с чёт/нечёт неделями
- Конфликт-детектор
- Автогенерация расписания
- Доступность преподавателей
- Совмещённые пары

#### Демо: Спортивные секции (НОВОЕ)
- 3 роли: Спортсмен, Тренер, Администратор
- 9 видов спорта, 11 тренеров, 10 площадок
- Расписание тренировок
- Посещаемость и аналитика
- Автогенерация расписания
- Зелёный акцент (#10b981)

#### Демо: Клубы и мероприятия (НОВОЕ)
- 3 роли: Участник, Организатор, Администратор
- 9 мероприятий, 11 организаторов, 10 помещений
- Календарь событий
- Бронирование ресурсов
- Статистика вовлечённости
- Оранжевый акцент (#f59e0b)

### Технологии
- Next.js 16 + App Router + TypeScript 5
- Tailwind CSS 4 + shadcn/ui
- Zustand для управления состоянием
- Статический экспорт для GitHub Pages

### Структура навигации
- Главная → 3 карточки-демо → Кнопка "Открыть демо"
- В каждом демо: кнопка "назад" (←) для возврата на главную
- Внутри демо: переключение ролей через кнопки в хедере
