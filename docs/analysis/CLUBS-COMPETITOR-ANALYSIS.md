## EXECUTIVE SUMMARY

The club and event management software market is fragmented across several categories:
- **Community platforms** (Meetup, Clubhouse) — focus on member engagement
- **Ticketing platforms** (Eventbrite, Timepad, Qtickets) — focus on ticket sales
- **Scheduling tools** (Calendly, Acuity) — focus on appointment booking
- **Space management** (Nexudus, Cobot, Spacebring) — focus on coworking operations
- **Fitness/club CRMs** (impulseCRM, Sport Priority, ClubSystem) — focus on class schedules

**KEY FINDING**: No single platform combines **recurring event scheduling**, **member management**, **visual calendar**, and **Russian-language support** with a modern mobile-first UI. This is Planovo's whitespace.

## INTERNATIONAL COMPETITORS

### 1. MEETUP

**Target Audience**: Hobby groups, professional communities, interest-based clubs, social groups

**Key Features**:
- Group creation and management (1 group on Starter, unlimited on Standard/Pro)
- Event creation with RSVP system (limited to 2 events/month on Starter)
- Member dues collection via Stripe integration
- Group Discussions (primary community engagement feature)
- Event discovery feed for members
- Photo sharing for past events
- Co-organizer and assistant organizer roles
- Member messaging and announcements
- Automatic event reminders

**UI/UX Patterns**:
- **Mobile-first design** with bottom navigation (Home, Explore, Messages, Profile)
- Event calendar: **List view** (not grid calendar) sorted by date — shows upcoming events in a feed format
- Group page: Header image + group description + upcoming events list + past events
- Event page: Title, date/time, location map, attendee list (photos), description, RSVP buttons (Going/Not Going)
- "Explore" tab: Location-based discovery with category filters (Tech, Health, Arts, etc.)
- No week/month grid calendar view — only chronological list
- Notifications for new events and changes
- **UX pain point**: No visual calendar grid; events can only be viewed as a list; no recurring event creation (must manually re-create)

**Pricing Model**: Subscription-based (organizer pays)
- **Starter** (Free): 1 group, max 2 in-person events/month, 30 members max
- **Standard**: $29.99/month or $99.99/6 months (≈$16.67/mo)
- **Pro**: $55/group/month or $47/group/month (6-month plan)
- Member dues: Stripe processing fees apply

**Unique Selling Points**:
- Largest community discovery platform (60M+ members)
- Built-in audience discovery — new members find groups organically
- Social proof through attendee counts and group history
- Member dues system for monetization

**What They Do WELL**:
- Community discovery and organic growth
- Simple event creation flow
- Member engagement through Discussions
- Mobile app is polished and intuitive
- Free Starter plan lowers barrier to entry

**What They Do POORLY** (🔥 Planovo Opportunities):
- **NO recurring/scheduled events** — must manually create each event
- **NO visual calendar grid** — only list view for events
- **NO resource/room booking** — can't book spaces or equipment
- **NO ticketing for paid events** — only member dues (no per-event tickets)
- **NO waitlist management** — just RSVP yes/no
- **NO event templates** — every event created from scratch
- **NO attendance tracking** — doesn't track who actually showed up
- Expensive for small clubs ($30/mo for basic)
- Limited customization of group pages
- Paywalling features (recently removed free tier for most organizers)

### 2. EVENTBRITE

**Target Audience**: Event organizers, venues, festivals, conferences, workshops, class providers

**Key Features**:
- Event creation with rich landing pages
- Multi-tier ticketing (General, VIP, Early Bird, etc.)
- Discount codes and promotional tools
- Embedded checkout widget for external websites
- Real-time sales dashboard
- Attendee check-in app (separate Organizer app)
- Email marketing tools (tiered: 2K/6K/10K emails/month)
- Social media integration and ad campaigns
- AI-powered event setup
- Custom order forms and attendee questionnaires
- Reserved seating with interactive seat maps
- "Eventbrite Ads" for promotion
- Donation tickets for nonprofits

**UI/UX Patterns**:
- **Organizer Dashboard**: Clean sidebar navigation with sales analytics, attendee management
- **Event creation wizard**: Step-by-step flow (Details → Tickets → Settings → Design)
- **Calendar view**: Events displayed in a **list/schedule format**, not a visual calendar grid
- **Attendee management**: Table view with search, filters, check-in status
- **Mobile Organizer app**: Check-in scanner, real-time sales, push notifications
- **Consumer app**: Event discovery with date/time filters, location-based, category browsing
- Dynamic dashboard visualizing ticket sales over time
- No built-in recurring event support — each event is standalone

**Pricing Model**: Free to publish + per-ticket fees
- **Free events**: No fees at all
- **Paid events (Flex plan)**: 3.7% + $1.79 service fee per ticket + 2.9% payment processing per order
- **Email marketing**: $15/mo (2K emails), $50/mo (6K), $100/mo (10K)
- **Pro plans**: Discounted fees for high volume + advanced marketing
- **Nonprofit discount**: 50% off Pro plans, discounted donation ticket fees

**Unique Selling Points**:
- World's largest event marketplace (90M+ ticket buyers)
- Zero upfront cost — fees only when tickets sell
- Comprehensive ticketing features (seat maps, discount codes, multi-tier)
- Strong analytics and reporting
- Embedded widget for selling tickets on your own site

**What They Do WELL**:
- Ticketing and payment processing (best-in-class)
- Event discovery marketplace
- Professional event landing pages
- Real-time sales dashboard
- Free tier for free events
- Extensive integrations (Mailchimp, Salesforce, etc.)

**What They Do POORLY** (🔥 Planovo Opportunities):
- **NO recurring event management** — designed for one-off events
- **NO member/community management** — no membership system, no community features
- **NO scheduling calendar** — no visual calendar for managing a schedule of classes/events
- **NO room/resource booking** — doesn't handle space allocation
- **Competitor events shown** on your event page (!)
- **Complex fee structure** — 3.7% + $1.79 + 2.9% adds up significantly
- **Monthly subscription add-ons** — emails, ads cost extra
- Not designed for clubs — focused on event promoters
- No RSVP without payment for paid events
- No member communication tools beyond email blasts

## UI/UX PATTERN ANALYSIS

### Event Calendar Patterns Across Platforms

| Pattern | Used By | Description |
|---------|---------|-------------|
| **List/Feed View** | Meetup, Eventbrite, Timepad | Events shown in chronological list — most common but least visual |
| **Monthly Grid Calendar** | Calendly, Acuity | Traditional calendar grid with available dates highlighted |
| **Weekly Schedule View** | Fitness CRMs, Nexudus | Time-block view showing daily schedule with room/resource allocation |
| **Date Picker Only** | Eventbrite (consumer) | Just a date selector, no full calendar |
| **Timeline/Schedule** | Event Rocks | Conference-style timeline with parallel tracks |

### Critical UX Gaps (Opportunities for Planovo)

1. **NO platform offers a beautiful visual calendar for managing recurring club events**
   - Meetup: List only
   - Eventbrite: No calendar management
   - Timepad: List only
   - Calendly: Grid calendar but only for 1:1 bookings

2. **NO platform combines scheduling + community + ticketing for clubs**
   - Ticketing platforms have no community
   - Community platforms have no scheduling
   - Scheduling tools have no community or ticketing

3. **NO Russian-language platform offers modern club scheduling**
   - Russian platforms focus on ticketing (Timepad, Qtickets)
   - Fitness CRMs are dated and niche-specific
   - No modern "calendar-first" approach

4. **Recurring events are universally poorly handled**
   - Most platforms require creating each event instance separately
   - No "every Tuesday at 7pm" type scheduling
   - Martin Fowler's recurring event pattern is not implemented in any consumer platform

5. **Mobile-first design is rare in management tools**

   - Click on calendar → fill in: name, time, type, room, capacity

   - One-tap booking with available spots shown
   - Waitlist indication ("3 on waitlist")
   - Personal schedule with upcoming bookings

   - QR code on member's phone
   - Or check off names from list
   - Automatic membership deduction

5. **Club Dashboard**
   - Today's schedule at a glance
   - Upcoming classes with fill rates (e.g., "Yoga: 8/12 spots")
   - Quick stats: this week's attendance, revenue

### 🎨 UI/UX Principles for Demo

- **Mobile-first** — most club organizers manage from their phone
- **Calendar-first** — the calendar IS the app, not a secondary view
- **Minimal clicks** — create an event in 2-3 taps, not 10 steps
- **Visual, not textual** — show availability with colors and shapes, not numbers
- **Russian language** — full localization, not just translation
- **Modern aesthetic** — clean whitespace, rounded corners, subtle animations (like Calendly's polish)
- **Contextual actions** — long-press/click on event for edit, tap for details

## CONCLUSION

The market has a clear gap: **no modern, calendar-first, Russian-language platform for club and event scheduling**. 

- Ticketing platforms (Timepad, Qtickets, Eventbrite) focus on selling tickets, not managing schedules
- Community platforms (Meetup) focus on discovery, not operational management
- Scheduling tools (Calendly, Acuity) focus on 1:1 appointments, not group classes
- Space management (Nexudus, Cobot) focuses on coworking, not events
- Fitness CRMs are dated and niche-specific

**Planovo's positioning**: "The beautiful calendar that runs your club" — visual scheduling + member management + attendance tracking, designed for Russian clubs and event spaces.

The #1 feature to nail: **a gorgeous, intuitive, recurring-event calendar** that makes club owners say "finally, someone made this."
